# DAO Charter (Contribution-Only)

Purpose:
Coordinate contributors to build and maintain canon, schemas, and agent roadmaps.

This DAO does not:
- promise profits or returns
- market tokens as investments
- provide financial advice

Core process:
1) Proposal
2) Review window
3) Attestation log
4) Decision
5) Merge + version tag

Treasury principle:
Operational support only, documented and auditable.
